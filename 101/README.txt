МОЙ ИГРОВОЙ ШАБЛОН:
- Simple CSS - стили
- Phaser - графика  
- LocalForage - сохранения
- Day.js - время
- Polyfill - для старых Android
- Modernizr - проверка браузера

Как использовать:
1. Скачать все файлы
2. Открыть index.html в браузере
3. Начать делать свою игру!
