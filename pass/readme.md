# ⚡ NEON PASS 2026 — генератор безопасных паролей

**NEON PASS 2026** — это современный веб-генератор паролей с уникальным неоновым дизайном в стиле «военный/секретный объект». Проект полностью адаптирован под ПК, планшеты и мобильные устройства (Android/iOS 2026). Основная цель — создание сложных, но удобных в настройке паролей с визуальным контролем состава.

![Скриншот интерфейса](screenshot.png)

## 🚀 Возможности

- **7 категорий символов**:
  - 🔠 Заглавные латинские
  - 🔡 Прописные латинские
  - 🔤 Заглавные кириллические (по умолчанию выключены)
  - 📝 Прописные кириллические (по умолчанию выключены)
  - 🔢 Цифры
  - #️⃣ Простые спецсимволы (`!"№;%:?*()_+`)
  - 🔣 Сложные спецсимволы (`@#$^&*[]{}<>|/`)

- **Интерактивные иконки** — нажатие на иконку включает/отключает всю категорию. Активная подсвечивается неоном.

- **Система бустов** (увеличение веса категории):
  - 🔴 Красный буст +250% (до 2 одновременно)
  - 🟢 Зелёный буст +500% (только 1 на весь набор)

- **Счётчики символов** под каждой категорией показывают, сколько символов из неё попало в текущий пароль.

- **Гибкая длина пароля** — от 1 до 999 символов (поле ввода с шагом 1).

- **Интеллектуальная генерация** с ограничениями:
  - Запрет двух одинаковых символов подряд.
  - Запрет повтора через один символ (например, `aba` разрешено, `a?a` — нет, где `?` — любой).
  - Не более 3 одинаковых символов на каждые 24 символа (масштабируется пропорционально длине).

- **Мгновенная перегенерация** при любом изменении настроек (включение категории, буст, смена длины) или по кнопке «перезапуск».

- **Копирование в буфер** одной кнопкой (с уведомлением).

- **Тёмная неоновая тема** (никакого светлого фона, все элементы контрастны и читаемы).

- **Адаптивная вёрстка** — сетка категорий перестраивается под экраны от 320px до 4K.

## 🛠 Технологии

- Чистый **HTML5**, **CSS3** (без фреймворков)
- Нативный **JavaScript** (ES6)
- Криптостойкий генератор случайных чисел (`Math.random` с дополнительным перемешиванием)
- Flexbox / Grid для построения сетки
- Медиазапросы для всех типов устройств
