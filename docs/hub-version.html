Эта страница фиксирует контрольную версию HUB. Номер версии на главной странице, в документации и в changelog должен совпадать.

  • Создана отдельная тестовая витрина hub-v3.html.
  • Project Passport открывается рядом с карточкой, а не у края экрана.
  • Повторное нажатие на ⋯ закрывает Project Passport.
  • Теги работают как переключатели ON/OFF.
  • Связанные теги скрыты, пока не выбран первый тег.
  • Добавлена цветовая классификация тегов.
  • Качество отделено тёплым терракотово-золотым цветом.
  • Готовность, визуал и техника получили разные цвета HP-баров.
  • HTML, JavaScript и CSS получили отдельные цвета в языковых барах.
  • Добавлен GitHub-бейдж на карточку проекта.
  • Добавлена кнопка будущего сравнения проектов ⇄.
  • Заголовок SUNPOLE HUB стал кликабельным возвратом к началу.
  • Добавлена первая пасхалка ✶ hidden layer.
