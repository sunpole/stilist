SUNPOLE HUB v0.4.0-alpha

Кодовое имя: Project Passport / Compare Layer

Статус: visual-prototype / alpha / control point

Эта страница фиксирует контрольную версию HUB. Номер версии на главной странице, в документации и в changelog должен совпадать.

Контрольная точка: v0.4.0-alpha создана как отдельная тестовая страница hub-v4.html. Старые версии shop.html, hub.html, hub-v2.html и hub-v3.html сохранены.

Что входит в версию 0.4.0-alpha

  • Создана отдельная тестовая витрина hub-v4.html.
  • Добавлен рабочий режим сравнения проектов через кнопку ⇄.
  • Можно выбрать до 4 проектов для сравнения.
  • Выбранные карточки подсвечиваются.
  • Появляется нижняя панель сравнения.
  • Добавлена таблица сравнения: статус, категория, качество, готовность, визуал, техника, документация, теги, languages и связанные версии.
  • Сохранены функции v0.3.0: Project Passport возле карточки, toggle-теги, цветовые группы, GitHub-бейджи и пасхалка.
