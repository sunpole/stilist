# 📁 Idle Miner - Шахтёрский симулятор

## 🎮 ОПИСАНИЕ ИГРЫ
**Idle Miner** - это инкрементальная игра в жанре "кликер + idle". Игрок управляет шахтой, добывает различные виды руды и развивает свою горнодобывающую империю.

### 🎯 ЦЕЛЬ ИГРЫ
- Заработать как можно больше денег
- Открыть все виды шахт (уголь, медь, железо, золото)
- Нанять рабочих для автоматической добычи
- Улучшить шахту до максимального уровня

## ⚙️ ТЕХНОЛОГИЧЕСКИЙ СТЕК

### 🌐 ОСНОВНЫЕ БИБЛИОТЕКИ
```
📁 index.html          - Главный файл
📁 game.js            - Логика игры
```

### 🛠 ИСПОЛЬЗУЕМЫЕ БИБЛИОТЕКИ:

#### 1. **Simple CSS** 🎨
- **Назначение:** Готовые красивые стили
- **Что делает:** Автоматически делает интерфейс приятным
- **Как проверить:** Убрать ссылку - сайт станет "голым"

#### 2. **Phaser 3** 🎮  
- **Назначение:** Игровой движок
- **Что делает:** Отрисовывает анимированную шахту
- **Демонстрация:** Разные шахты разного цвета с анимациями

#### 3. **LocalForage** 💾
- **Назначение:** Сохранение игры
- **Что делает:** Сохраняет весь прогресс в браузере
- **Проверка:** Закрыть/открыть браузер - прогресс на месте

#### 4. **Day.js** ⏰
- **Назначение:** Работа со временем
- **Что делает:** Считает время игры, длительность сессии
- **Видно:** Таймер в статус-баре

#### 5. **Polyfill.io** 🔧
- **Назначение:** Совместимость со старыми браузерами
- **Что делает:** Добавляет современные функции в старые Android
- **Для кого:** Android 6.0-7.0, старые Safari

#### 6. **Modernizr** 🔍
- **Назначение:** Проверка возможностей браузера
- **Что делает:** Определяет что поддерживает браузер
- **Как увидеть:** Открыть Console в DevTools

## 🎯 ИГРОВАЯ МЕХАНИКА

### 💰 РЕСУРСЫ:
- **⛏️ Уголь** - базовый ресурс, добывается быстро
- **🔶 Медь** - в 5 раз дороже угля
- **⚙️ Железо** - в 20 раз дороже угля  
- **⭐ Золото** - в 100 раз дороже угля

### 👷 РАБОЧИЕ:
- Автоматически добывают руду
- Каждый рабочий +1 к добыче в секунду
- Цена найма растет с каждым рабочим

### 🏗️ УЛУЧШЕНИЯ:
- **Уровень шахты** - увеличивает добычу ВСЕЙ руды
- **Новые шахты** - открывают доступ к новой руде
